# 一，作用域

作用域，即变量（变量作用域又称上下文）和函数生效的区域或者集合

代码区块中的**变量和其他资源**的**可见性**由作用域决定：

~~~js
function example(){
    let a=114;
}
console.log(a)//会提示你a没有定义
~~~

这说明我们在全局无法获取到函数内的变量（除了闭包）

作用域可以分为：

>全局作用域
>
>函数作用域
>
>块级作用域

### 全局作用域

任何不在函数中或是大括号中声明的变量，都在全局作用域下，声明的变量可以任意访问

### 函数作用域

局部作用域，只能在函数内部访问

### 块级作用域

在大括号中使用let和const声明的变量存在于块级作用域中。在大括号外不能访问

# 二，词法作用域

静态作用域，变量被创建的时候就确定好了，而非执行阶段确定的。也就是说我们写好代码他的作用域就确定了，而不是调用函数才确定的，js遵循的就是词法作用域



# 三，作用域链

**当**在js中使用一个变量的时候，js引擎会先找当前作用域，如果没有就按个往上找直到全局作用域
