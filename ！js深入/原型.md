# 原型

每一个对象（函数也是对象）都有一个原型

### 什么是构造函数

~~~js
var Parent=function(){
}
//定义一个函数，那他只是一个普通的函数，不是构造函数
var instance=new Parent();
//创造了一个Parent构造函数的实例 instance
//这个时候parent就不是普通的函数了，他是一个构造函数。因为通过new关键字调用了它
~~~

### 什么是prototype属性

~~~js
//原型指向原型对象，原型对象存放这共享的属性方法

//函数特有的属性
var Parent=function(){
}
Parent.prototype.name="所有parent的实例都可以读取到我";
let p1=new Parent();
let p2=new Parent();
//这时候p1，p2是实例，有prototype原型存在，p1,p2就可以读取到这个name属性
p1.name
p2.name
~~~

### 什么是_proto_

proto是隐式原型，指向构建他的实例函数的原型对象

~~~js
//函数特有的属性
var Parent=function(){
}
Parent.prototype.name="所有parent的实例都可以读取到我";
let p1=new Parent();
let p2=new Parent();
//这时候p1，p2是实例，有prototype原型存在，p1,p2就可以读取到这个name属性
p1.name
p2.name

p1.__proto__===parent.prototype //true
//proto存在的意义是为原型链查找提供方向
~~~

### 查找name元素总结:

1,p1对象身上有没有name属性

2,p1.____proto____(Parent.prototype)

3,Object.prototype上有没有name属性

（ps：parent.prototype.__proto__===Object.prototype）

Object.prototype便是**js原型链的顶点**的起点

所有对象，包括构建函数的原型对象，都源自于Object.prototype

### 原型链基本概念

每个对象都有一个原型，并从原型身上继承属性和方法。

原型本身也是一个对象，也有自己的原型，形成一个链式结构
