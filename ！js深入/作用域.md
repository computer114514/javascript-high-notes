# 词法作用域和动态作用域

作用域指程序源代码定义变量的区域

## 静态作用域

js采用的是静态作用域

~~~js
let value=1;

function foo(){
    console.log(value);
}

function bar(){
    let value=2;
    foo();
}

bar();
//最后打印出来的是1
~~~

要打印value之时，先从foo内部寻找是否有value，如果没有，**那就根据书写的位置，查找上面一层代码**，这就是全局作用域，也就会打印1

## 动态作用域

如果是动态作用域，那么也会先看foo内部value变量有无，如果没有，**那会寻找调用函数的作用域**，也就是bar函数内部寻找value变量，最后打印2
